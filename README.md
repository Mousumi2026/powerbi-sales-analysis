# üçï Pizza Sales Dashboard - Power BI Project

This repository contains a **Power BI dashboard** for analyzing pizza sales data.  
The dashboard provides insights into **revenue, orders, pizza types, and sizes**, helping businesses make data-driven decisions.

---

## üìä Overview    


The dashboard includes:

- **Key Metrics (KPI Cards)**  
  - Total Revenue: 817.86K  
  - Total Pizza Sold: 50K  
  - Total Orders: 21K  
  - Average Pizza Per Order: 2.32  
  - Average Order Value: 0.03  

- **Top & Bottom Pizza Insights**  
  - Revenue, Orders, and Quantity by **Top 5 and Bottom 5 pizzas**  
  - Identifies high-performing and low-performing items for marketing or operational decisions

- **Trends & Patterns**  
  - Daily order trends (highest orders on **Friday**)  
  - Monthly order trends (peak month: **July**)  

- **Sales Breakdown**  
  - % of sales by pizza category (Classic, Supreme, Veggie, Chicken)  
  - % of sales by pizza size (Small, Medium, Large, X-Large)  
  - Total pizza sold by category  

- **Insights Section**  
  - Highlights key business findings:
    - Peak day for orders  
    - Peak month for sales  
    - Top-selling pizza category  
    - Most popular pizza size
